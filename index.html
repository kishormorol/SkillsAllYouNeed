About · v2.0

An open registry of usable AI skill definitions — Claude Code .md skills, CLAUDE.md templates, Claude.ai skill uploads, ChatGPT custom instructions, and more. Composed in plain HTML & CSS without a framework. v1.x: capability directory with filters, deep‑linking, keyboard shortcuts, favorites, Export JSON. v2.0: redesigned as a skills marketplace; removed academic framing.
